# Core

This folder contains all code related to technical Windows mechanics.

## Focus manager

> focus_manager.py

Contains code to focus a window

## Input simulator

> input_simulator.py

Provides a set of utilities to manage hotkey presses and other keyboard actions.

## Keyboard monitor

> keyboard_monitor.py

Service that registers hotkeys to listen to with callbacks while the application is running.

## System tray

> system_tray.py

Provides a set of utilities related to system tray icon.

## Windows manager

> window_manager.py

Provides a set of utilities to manage active "Dofus Retro" windows. This is used to keep a track of all characters for example.
